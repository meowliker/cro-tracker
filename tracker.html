Minding Art CRO Tracker
One-time setup — takes about 5 minutes
1
Create a Google Sheet
Go to sheets.google.com, create a new blank spreadsheet, name it Minding Art CRO Tracker.
2
Open Apps Script
In the spreadsheet: Extensions → Apps Script. Delete all default code.
3
Paste Code.gs
Open Code.gs from the zip, copy everything, paste it in. Save (Ctrl+S).
4
Deploy as Web App
Click Deploy → New deployment → Web app.
• Execute as: Me
• Who has access: Anyone
Click Deploy, allow permissions, copy the URL.
5
Paste your Apps Script URL below
It looks like:
https://script.google.com/macros/s/AKfy.../exec
Please enter a valid Apps Script URL
After connecting, share this HTML file with your team — everyone connects to the same Google Sheet. All changes are logged in the Activity_Log tab of your spreadsheet.
